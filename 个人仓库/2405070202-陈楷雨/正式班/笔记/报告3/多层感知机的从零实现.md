# 多层感知机的从零实现

## 1. 导入必要的库并读取数据集

```python
import sys
sys.path.append('..')  # 向上1级到正式班目录
from my_utils import *


import torch
from torch import nn
from d2l import torch as d2l
batch_size = 256
train_iter, test_iter = d2l.load_data_fashion_mnist(batch_size)
```

## 2. 定义模型参数

实现一个具有单隐藏层的多层感知机，包含256个隐藏单元。

```python
num_inputs, num_outputs, num_hiddens = 784, 10, 256
w1 = nn.Parameter(
    torch.randn(num_inputs, num_hiddens, requires_grad=True) * 0.01
)  # 这里是随机选取，如果是0的话，就是线性的了
b1 = nn.Parameter(torch.zeros(num_hiddens, requires_grad=True))
w2 = nn.Parameter(
    torch.randn(num_hiddens, num_outputs, requires_grad=True) * 0.01
)
b2 = nn.Parameter(torch.zeros(num_outputs, requires_grad=True))

params = [w1, b1, w2, b2]
```

## 3. 实现ReLU激活函数

```python
def relu(x):
    a = torch.zeros_like(x)
    return torch.max(x, a)
```

## 4. 定义模型

```python
def net(x):
    x = x.reshape((-1, num_inputs))  # 这里的-1是自动张开，输入的形状(32, 28, 28)，32是批量，reshape就是(32，784)，每行是一个样本，一个图片就是一个行向量
    h = relu(x @ w1 + b1)  # @是数学中矩阵的乘法，满足前一列对于后一行
    return h @ w2 + b2

loss = nn.CrossEntropyLoss()
```

## 5. 定义优化算法

使用随机梯度下降优化器，学习率为0.1。

```python
num_epochs, lr = 10, 0.1
updater = torch.optim.SGD(params, lr=lr)
```

## 6. 训练模型

多层感知机的训练过程与softmax回归的训练过程完全相同。

```python
train_ch3(net, train_iter, test_iter, loss, num_epochs, updater)
```

## 7. 小结

- 从零开始实现多层感知机需要手动定义模型参数、激活函数和前向传播过程。
- 与简洁实现相比，从零实现能更深入地理解模型内部工作原理。
- 通过手动实现可以更好地掌握梯度计算和参数更新过程。
- 虽然代码量较大，但对于学习深度学习原理非常有帮助。

## 8. 思考

1. 从零实现虽然繁琐，但能加深对神经网络前向传播和反向传播的理解。
2. 手动初始化参数时需要注意尺度，过大的初始值可能导致训练不稳定。
3. 实现ReLU激活函数时需要注意使用`torch.max`而不是`max`函数，以保持张量运算。
4. 与简洁实现相比，从零实现的训练效果相近，但代码可维护性较差，适合教学和原理学习。
5. 在实际项目中，通常使用框架提供的高级API以提高开发效率。
