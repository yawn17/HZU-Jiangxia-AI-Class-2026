# 多层感知机的简洁实现

## 1. 导入必要的库并读取数据集

```python
import torch
from torch import nn
from d2l import torch as d2l

import sys
sys.path.append('..')  # 向上1级到正式班目录
from my_utils import *
```

## 2. 定义模型网络结构（ReLU激活函数）

隐藏层包含256个隐藏单元，使用ReLU激活函数。

```python
# ReLU激活函数
net = nn.Sequential(nn.Flatten(),
                    nn.Linear(784, 256),
                    nn.ReLU(),
                    nn.Linear(256, 10))
```

## 3. 初始化模型权重

使用正态分布初始化线性层的权重，标准差设为0.01。

```python
def init_weights(m):
    if type(m) == nn.Linear:
        nn.init.normal_(m.weight, std=0.01)

net.apply(init_weights)
```

## 4. 定义损失函数和优化算法

使用交叉熵损失函数和学习率为0.1的随机梯度下降优化器。

```python
batch_size, lr, num_epochs = 256, 0.1, 10
loss = nn.CrossEntropyLoss()
trainer = torch.optim.SGD(net.parameters(), lr=lr)
```

## 5. 训练模型（ReLU）

加载Fashion-MNIST数据集并开始训练。

```python
train_iter, test_iter = d2l.load_data_fashion_mnist(batch_size)
train_ch3(net, train_iter, test_iter, loss, num_epochs, trainer)
```

训练结果：
```
(0.3803981584310532, 0.86595, 0.8493)
```
（输出为最终训练损失、训练集准确率、测试集准确率）

## 6. 使用Sigmoid激活函数

将ReLU激活函数替换为Sigmoid激活函数进行比较。

```python
# Sigmoid激活函数
net = nn.Sequential(nn.Flatten(),
                    nn.Linear(784, 256),
                    nn.Sigmoid(),
                    nn.Linear(256, 10))

def init_weights(m):
    if type(m) == nn.Linear:
        nn.init.normal_(m.weight, std=0.01)

net.apply(init_weights)

batch_size, lr, num_epochs = 256, 0.1, 10
loss = nn.CrossEntropyLoss()
trainer = torch.optim.SGD(net.parameters(), lr=lr)

train_iter, test_iter = d2l.load_data_fashion_mnist(batch_size)
train_ch3(net, train_iter, test_iter, loss, num_epochs, trainer)
```

## 7. 使用Tanh激活函数

将激活函数替换为Tanh进行比较。

```python
# Tanh激活函数
net = nn.Sequential(nn.Flatten(),
                    nn.Linear(784, 256),
                    nn.Tanh(),
                    nn.Linear(256, 10))

def init_weights(m):
    if type(m) == nn.Linear:
        nn.init.normal_(m.weight, std=0.01)

net.apply(init_weights)

batch_size, lr, num_epochs = 256, 0.1, 10
loss = nn.CrossEntropyLoss()
trainer = torch.optim.SGD(net.parameters(), lr=lr)

train_iter, test_iter = d2l.load_data_fashion_mnist(batch_size)
train_ch3(net, train_iter, test_iter, loss, num_epochs, trainer)
```

## 8. 小结

- 使用PyTorch高级API可以更简洁地实现多层感知机模型。
- 通过比较不同激活函数（ReLU、Sigmoid、Tanh）可以观察其对训练效果的影响。
- `nn.Sequential`容器可以方便地堆叠网络层。
- 训练过程与从零实现相同，但代码更加清晰、易于维护。

## 9. 思考

1. 简洁实现利用了深度学习框架的高级抽象，减少了手动实现细节的工作量。
2. 通过框架内置的优化算法和损失函数，可以更好地调节参数。
3. 相比从零实现，简洁实现的代码更少、更易读，适合快速原型开发。
4. 不同激活函数对模型收敛速度和最终准确率有显著影响，需要根据任务选择合适的激活函数。
5. 在实际项目中，通常首选ReLU激活函数，因为它计算简单且能有效缓解梯度消失问题。
6. 我用了3种激活函数进行比对，ReLU激活函数的test_acc最高。
