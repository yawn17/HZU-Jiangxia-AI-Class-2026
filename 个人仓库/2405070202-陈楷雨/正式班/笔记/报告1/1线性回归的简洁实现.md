# 线性回归的简洁实现

## 1. 实验目标

使用 PyTorch 提供的高层 API，完成线性回归的简洁实现。与从零开始实现相比，这一版本把数据读取、模型定义、损失函数和优化器都交给框架处理，因此训练代码更短，也更接近真实项目中的写法。

## 2. 数据集构造

先用一个带噪声的线性模型生成合成数据：

$$
\mathbf{y} = \mathbf{X}\mathbf{w} + b + \epsilon
$$

其中：

- $\mathbf{w} = [2, -3.4]^\top$
- $b = 4.2$
- $\epsilon$ 是噪声项

代码中使用 `d2l.synthetic_data` 生成 1000 个样本，并查看一个样本的特征和标签。

## 3. 数据读取

使用 `TensorDataset` 和 `DataLoader` 构造数据迭代器：

```python
def load_array(data_arrays, batch_size, is_train=True):
	dataset = data.TensorDataset(*data_arrays)
	return data.DataLoader(dataset, batch_size, shuffle=is_train)
```

作用：

- 把特征和标签打包成数据集
- 按批次读取小批量数据
- 训练时自动打乱样本顺序

## 4. 模型定义

使用 `nn.Sequential` 和 `nn.Linear` 直接定义线性模型：

```python
from torch import nn
net = nn.Sequential(nn.Linear(2, 1))
```

这里表示：

- 输入维度是 2
- 输出维度是 1
- 模型形式就是 $\hat{y} = Xw + b$

## 5. 参数初始化

线性层的参数需要手动初始化：

```python
net[0].weight.data.normal_(0, 0.01)
net[0].bias.data.fill_(0)
```

一般做法是：

- 权重初始化为均值 0、标准差 0.01 的正态分布
- 偏置初始化为 0

## 6. 损失函数

这里使用均方误差损失 `MSELoss`：

```python
loss = nn.MSELoss()
```

它衡量的是预测值和真实值之间的平方误差，适合回归任务。

## 7. 优化算法

使用随机梯度下降 `SGD`：

```python
trainer = torch.optim.SGD(net.parameters(), lr=0.03)
```

说明：

- `net.parameters()` 包含模型的所有可学习参数
- `lr` 是学习率，决定每次参数更新的步长

PyTorch 中梯度默认会累加，所以每轮更新前需要先清零梯度。

## 8. 训练过程

训练循环的核心步骤是：

```python
num_epochs = 3
for epoch in range(num_epochs):
	for x, y in data_iter:
		l = loss(net(x), y)
		trainer.zero_grad()
		l.backward()
		trainer.step()
	with torch.no_grad():
		l = loss(net(features), labels)
		print(f'epoch {epoch + 1}, loss {l:f}')
```

可以理解为：

1. 前向传播得到预测值
2. 计算损失
3. 清空旧梯度
4. 反向传播计算当前梯度
5. 用优化器更新参数
6. 统计整体验证损失

## 9. 梯度清零的位置

这里的 `trainer.zero_grad()` 必须放在 `backward()` 之前。<br>
- 在从零开始实现时，我们是在训练过程的最后清零的，和我们这个在训练的最开始清零目的相同，也可以以在从零开始那里的sgd函数不清零，在外层的for循环里清零，效果也是一样的。就是将最后清零转化为在训练的最开始清零。


原因是：

- PyTorch 的梯度默认会累加
- 如果不清零，上一轮的梯度会混到下一轮里
- 先清零，再反向传播，得到的才是当前 batch 的梯度

这也是 PyTorch 里最常见的训练写法。

## 10. 结果观察

训练完成后，可以把学到的参数和真实参数比较：

```python
print(f'w的估计误差: {true_w - net[0].weight.data.reshape(true_w.shape)}')
print(f'b的估计误差: {true_b - net[0].bias.data}')
```

如果训练正常，误差应该比较小，说明模型已经学到了接近真实的线性关系。

## 11. 学习率的影响

学习率太小会导致收敛很慢，学习率太大则可能导致震荡甚至发散。

因此，训练效果不仅取决于模型结构，也和优化超参数密切相关。

## 12. 小结

这一节展示了如何借助 PyTorch 快速完成线性回归：

- 用 `DataLoader` 读取数据
- 用 `nn.Linear` 定义模型
- 用 `MSELoss` 计算损失
- 用 `SGD` 更新参数

相比从零开始实现，这种方式更简洁，也更贴近实际工程写法。
